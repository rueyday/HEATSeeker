Overview

HEATSeeker is a smart-glasses + rescue robot project that detects heat signatures using an AMG8833 thermal camera. The robot streams real-time 8×8 thermal frames to a head-mounted display inside custom glasses, allowing situational awareness during remote operation.

System Architecture

The controller sends navigation commands to the robot, and the robot transmits thermal data to the glasses using XBee. The glasses overlay heat-map visuals directly in the user's field of view.

Components

In-Lab Components

  • STM32 Microcontroller
  • AMG8833 Thermal Camera
  • XBee Wireless Modules
  • OLED Display for HUD
  • DC Motors & Motor Driver
  • Pushbuttons & Switches

Possible Additions

  • IMU-based head tracking
  • LiPo battery module
  • TOF distance sensor (VL53L1X)
  • Custom wearable PCB

Reach Goals

  • Custom PCB for the smart-glasses module
  • SD-card logging of thermal frames
